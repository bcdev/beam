Import ENVISAT

Import ENVISAT (MERIS,AATSR,ASAR) Product

This option allows to import ENVISAT products into VISAT.

The supported ENVISAT product types are:

  • MERIS L1b/L2 RR/FR/FRG/FSG products
  • AATSR TOA L1b and NR L2
  • ASAR products

NOTE:
When using MERIS FRG/FSG products they are, by default, imported with a tie-point based geo-coding. The per-pixel geo-information is not considered. To make use of it, you can attach a Pixel Geo-Coding. If you want BEAM to load per-pixel geo-information on import, which is useful for processing, you can enable the beam.envisat.usePixelGeoCoding property in the beam.config file. This file is located in the config directory of your BEAM installation.
